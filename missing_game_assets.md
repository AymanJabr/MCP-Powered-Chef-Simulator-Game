## Missing CookingActionTypes

The following `CookingActionType` values are used in `recipes.json`'s `cookingSteps` but are not defined in the `CookingActionType` type alias in `src/types/models.ts`:

- `type: 'toast'` (Used in recipe_burger_cheese_double, recipe_burger, recipe_hotdog, recipe_sandwich, recipe_sub, recipe_waffles_with_honey, recipe_waffle_with_sauce)
- `type: 'mix'` (Used in recipe_chocolate_milk_shake_root_beer, recipe_cocktail, recipe_frappe, recipe_milkshake_chocolate, recipe_milkshake_desert_chocolate_sweet, recipe_popsicle_chocolate)

Additionally, some comments imply actions not covered by existing types or equipment:
- `action: 'blending'` (Implied by missing `equipment_blender`)
- `action: 'juicing'` (Implied by missing `equipment_manual_juicer`)
- `action: 'brewing'` (Implied by missing `equipment_coffee_maker`)
- `action: 'freezing'` (Implied by missing `equipment_freezer` / `equipment_ice_cream_maker`)
- `action: 'crushing'` (Implied for cookies in recipe_milkshake_desert_chocolate_sweet)
- `action: 'assembling_skewers'` (Implied by recipe_skewer_vegetables)
- `action: 'carbonating'` (Implied by soda recipes)
- `action: 'steaming'` (While `boil` exists, dedicated `steam` might be useful if `equipment_steamer` is added; `boil` in `equipment_rice_cooker` is a placeholder for recipe_steamed_buns)
- `action: 'rolling_sushi'` (Implied by sushi recipes)
- `action: 'making_waffles'` (Implied by waffle recipes and missing `equipment_waffle_iron`)
- `action: 'glazing'` (Implied in comments for recipe_ribs_cooked)
- `action: 'laminating_dough'` (Implied in comments for recipe_croissant)
- `action: 'resting_meat'` (Mentioned in comments for recipe_steak_cooked)
- `action: 'thickening_sauce_or_pudding'` (Implied by missing thickeners like cornstarch/flour)
- `action: 'garnishing'` (Implied by comments, but not a formal step type)
- `action: 'dispensing_soda'` (Implied for soda recipes with `equipment_beverage_dispenser`)

**Note:** Some "Missing Ingredients" like "salt" or generic "herbs" might be considered universally available or abstracted. The list above includes items explicitly mentioned as missing or heavily implied by recipe context and comments.
